## Kid learning

### Start the app
`cargo tauri dev`

### Rust version
`rustc --version`

### upgrade rust
`rustup update`

### upgrade tauri cli
modify Cargo.toml => doesn't work
`cargo install tauri-cli`

### upgrade cargo packages
go to src-tauri
`cargo update`

### check tauri version
`cargo tauri info`


### upgrade all packages
under direct project
`yarn upgrade @tauri-apps/cli @tauri-apps/api --latest`
`yarn upgrade --latest`
